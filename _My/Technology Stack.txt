

ТЕХНОЛОГИИ
1. Laravel - php фреймворк для backend: роуты, контроль авторизаций, аутентификаций, работа с базой данных, контроль ролей и привилегий, контроль путей (доступность относительно ролей)
2. Vue - JS фреймворк для frontend: реактивные страницы (переходы без перезагрузки страницы)
3. GitHub - хранение исходников, контроль версий, автодеплой.
4. CASL
5. spatie - реализация системы ролей + точечные привилегии.
6. Sanctum - автоматическая аутентификация по cookie для SP и токен-аутентификация для api для сторонних приложений (и для SPA тоже).
8. Axios - отправка команд из SPA front (Vue) в backend (Laravel).
9. Nested Set - организация базы данных для быстрого поиска вложенных списков.
10. feather-icons - иконки.
11. Vuexy - bootstrap шаблон дизайна.
