


ТЕХНОЛОГИИ
1. Laravel - php фреймворк для backend: роуты, контроль авторизаций, аутентификаций, работа с базой данных, контроль ролей и привилегий, контроль путей (доступность относительно ролей)
2. Vue - JS фреймворк для frontend: реактивные страницы (переходы без перезагрузки страницы)
3. GitHub
4. CASL
5. spatie
6. Sanctum
7. GitHub
8. Axios - отправка команд между front и back
9. Nested Set - организация базы данных для быстрого поиска пложенных списков
10. feather-icons - иконки


УСТАНОВКА
- установить пхпшторм, xammp
- на мак установить homebrew
- с помощью homebrew установить node (+npm)
- залогиниться в браузере в гитхаб
- в пхпшторм подкачать из гитхаб проект
- composer install
- php artisan key:generate
- npm install
- cd frontend
- npm install
- cd ..
- npm run dev
- php artisan serve
- npm run watch
- php artisan migrate
- php artisan db:seed
- php artisan migrate:fresh --seed --seeder=PermissionsDemoSeeder
- php artisan db:seed
